# Snow Tmux

Based on [Nord Tmux](https://github.com/arcticicestudio/nord-tmux).

Checkout their ReadMe. It should work almost the same.

## Install

### tpm

Add to your `~/.tmux.conf`:

```bash
set -g @plugin 'mcchrish/snow-tmux'
```
